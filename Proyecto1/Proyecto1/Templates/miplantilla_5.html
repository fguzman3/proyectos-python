En esta página aplicaré IF condicional y ademas a nivel de backend trabajare con loader, que es una clase del modulo Template de django

  1. El modulo loader permite abrir un flujo de datos externos, sin tener que pasarle manualmente la ruta de donde se origina el dato.
  2. Del modulo loader usaremos la clase get_template, que permite hacer la acción indicada
  3. se debe importar asi: from django.template.loader import get_template en la parte de las vistas (views.py)
  4. Se debe verificar en archivo settings.py la ruta donde quedaran los templates.
